Guía Rápida de Comandos Git

git init

inicializa un nuevo repositorio Git local en el directorio actual

git status

Muestra el estado actual del repositorio,archivos modificados y norastreados.

git add .

agrega todos los archivos al área de preparación (staging)

git commit -m "mensaje"

Guarda los cambios agregados con un mensaje descriptivo

git push

Envía los commits locales al repositorio remoto

git pull

Obtiene y fusiona los cambios del repositorio remoto
